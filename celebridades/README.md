# Fotos de Referência para Reconhecimento Facial

Coloque 3-5 fotos de cada celebridade nas pastas correspondentes.

## Estrutura:
```
celebridades/
├── gretchen/    ← coloque fotos da Gretchen aqui
├── patixa/      ← coloque fotos da Patixa aqui
└── malevola/    ← coloque fotos da Malévola aqui
```

## Dicas:
- Use fotos com rosto visível e claro
- Diferentes ângulos ajudam na precisão
- Formatos: jpg, jpeg, png, webp

## Depois de adicionar as fotos:
```bash
./scripts/add-celebrity.sh gretchen
./scripts/add-celebrity.sh patixa
./scripts/add-celebrity.sh malevola
```
